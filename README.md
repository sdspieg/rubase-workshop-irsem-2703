# RuBase Methods Workshop - The Fletcher School

**AI-Augmented Research Methods**
March 11-13, 2026 | 5:30-8:30 PM

## 🎯 Workshop Overview

This is a comprehensive 3-day workshop introducing participants to AI-augmented research methods using the RuBase/StratBase approach. The workshop features a unified web application with structured learning paths, interactive tools, and hands-on exercises.

## 🌐 Live Demo

🔗 **[View Workshop App](https://sdspieg.github.io/rubase-workshop-fletcher-2603/)**

## 📐 Architecture

### Three-Tier Structure

1. **DAYS** - Timeline
   - Day 1: DISCOVER (Finding What You've Been Missing)
   - Day 2: FRAME (Building Rigorous Analytical Pipelines)
   - Day 3: ANALYZE (From Frameworks to Findings)

2. **MODULES** - Sessions within each day
   - Welcome & Setup
   - RuBase Methodology
   - Bibliometrics
   - Corpus Building
   - Taxonomy Development
   - LLM Classification

3. **SECTIONS** - Deep-dive content within modules
   - OpenAlex Explorer (20 interactive slides)
   - Dashboard Analytics
   - The Lens Walkthrough
   - API Tutorials
   - Hands-on Exercises

## 🚀 Features

### Navigation System
- **Progressive Learning Path**: Follow the workshop schedule
- **Explorer Mode**: Jump to any content directly
- **Breadcrumb Navigation**: Always know where you are
- **Progress Tracking**: See your completion status

### Content Integration
- **OpenAlex Module**: Complete bibliometric analysis toolkit
- **Interactive Dashboard**: Visualize 6,329 research papers
- **Tech Help**: Built-in troubleshooting guide
- **Resource Library**: All documentation in one place

### Responsive Design
- Works on laptops, tablets, and mobile devices
- Collapsible sidebar for mobile viewing
- Optimized for presentation and self-study

## 🎓 Learning Outcomes

By completing this workshop, participants will:
- Build a complete research corpus using OpenAlex
- Develop custom taxonomies for their domain
- Apply LLM classification to large datasets
- Create interactive visualizations
- Master the complete RuBase methodology

## 📊 Day-by-Day Breakdown

### Day 1: DISCOVER
- Introduction to RuBase methodology
- Bibliometric fundamentals with OpenAlex
- Finding hidden literature (56% problem)
- Building your first corpus

### Day 2: FRAME
- Taxonomy theory and design
- Multi-dimensional categorization
- LLM-assisted development
- Validation techniques

### Day 3: ANALYZE
- Classification at scale
- Visualization strategies
- Research output generation
- Future directions
