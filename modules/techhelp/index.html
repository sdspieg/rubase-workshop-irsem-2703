Troubleshooting Guide & Common Solutions

🔧 Common Issues & Solutions

Can't see slide content / Content is cut off
  • Try refreshing the page (F5 or Ctrl+R)
  • Zoom out slightly (Ctrl+- or Cmd+-)
  • Use Chrome or Firefox for best compatibility
  • Check if you're in full-screen mode (F11 to toggle)
Can't access Dropbox folder
  • Check your email for the shared link invitation
  • Make sure you're logged into Dropbox
  • Contact instructors if you haven't received the invitation
  • Try opening the link in an incognito/private window
API Key not working
  • Verify you've registered at openalex.org
  • Check that you've copied the entire API key
  • Ensure there are no extra spaces before/after the key
  • API keys are case-sensitive - copy exactly as shown
Navigation buttons not working
  • Use arrow keys (← →) to navigate between slides
  • Press ESC to return to the main menu
  • Click the slide dots at the bottom for direct navigation
  • Clear browser cache (Ctrl+Shift+Delete)

🔑 API Setup Guide

OpenAlex API Setup
  1. Go to openalex.org
  2. Click "Get API Key" or "Sign Up"
  3. Fill in your email and affiliation
  4. Check your email for the API key
  5. Save the key in a secure location
The Lens API Setup
  1. Visit lens.org
  2. Create a free account
  3. Go to Account Settings → API
  4. Request API access (may take 24h)
  5. Copy your token once approved
