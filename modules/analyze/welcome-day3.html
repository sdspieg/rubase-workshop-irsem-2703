CLI Integration: Your AI Research Assistant

The Game Changer: Voice Control

Don't type everything - speak your commands!

How to Send Voice Messages to CLI

🍎 macOS

Press Fn key twice → Dictation mode

Speak naturally → Text appears in terminal → Press Enter

🪟 Windows

Press Win + H → Voice typing activates

Dictate your command → Windows converts to text → Send to CLI

🐧 Linux

Install tools: nerd-dictation or whisper.cpp

Configure hotkeys → Speak → Text appears

📱 Universal Alternative

Use phone's voice-to-text → Copy → Paste to terminal

Live Demo: Watch me control Claude Code with voice!

LLMs as a Jury: Universal Principle

The Consensus Approach

For EVERY important decision:

  1. Get multiple independent solutions
  2. Have LLMs deliberate amongst themselves
  3. Extract consensus recommendations
  4. You make the final decision

Applications

  • Code solutions
  • Research interpretations
  • Data analysis approaches
  • Taxonomy development
  • Classification decisions

Benefits

  • Reduces hallucinations
  • Catches edge cases
  • Multiple perspectives
  • Higher confidence
  • Better decisions
⚠️ Cost Warning: Running multiple LLMs for consensus can become expensive quickly. Use judiciously for critical decisions, not routine tasks.

Example: 3 LLMs analyzing Ottoman Bank papers → 95% agreement on categories

Discovery Methods Overview

Bibliographic vs Fulltext

Bibliographic (OpenAlex):

  • Metadata: titles, abstracts, authors
  • Citations and references
  • Fast and comprehensive
  • 596 Ottoman Bank papers found!

Fulltext:

  • Complete article content
  • Deep textual analysis
  • Often behind paywalls
Corpora & Databases

Pre-built Collections:

  • Domain-specific databases
  • Government repositories
  • Organization archives
  • Historical collections

Official Sources:

  • Central bank archives
  • Ministry publications
  • International organizations

Key Insight: Start with bibliographic discovery

Then selectively acquire fulltext for deep analysis
